# Gauri Bhonsle Portfolio

Personal portfolio site built with React, TypeScript, Vite, Tailwind CSS, and GSAP.

## Live Site

If deployed, this project is intended to be hosted on a public domain such as `gauribhonsle.in` or `gauribhonsle.com` and indexed by Google.

## Features

- Animated hero section with layered visuals
- About, Projects, Experience, and Footer sections
- Social links to GitHub and LinkedIn
- Responsive layout for desktop and mobile
- Project cards with repository and live-demo links

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- GSAP
- Lucide React

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment Notes

- Update the page title and metadata before deploying.
- Connect the app to a public domain for search engine indexing.
- Submit the deployed URL to Google Search Console after launch.
